OpenAI ChatGPT API介绍

OpenAI ChatGPT API地址：https://platform.openai.com/docs/api-reference/

OpenAI ChatGPT API是OpenAI开放的可以调用GPT模型的接口（API：Application Programming Interface 应用程序编程接口），包括GPT-3.5 API、GPT-4.0 API等，所以OpenAI ChatGPT API更准确的叫法应该是OpenAI API。但是OpenAI API中的GPT-3.5模型的API就是ChatGPT目前使用的模型，并且GPT-3.5 API也是用的最多的API，因此也有人称之为ChatGPT API。

有了OpenAI ChatGPT API我们就可以基于基于ChatGPT做二次开发，例如聊天机器人、智能客服、语音识别、自然语言生成等，一般来说，在其他平台或软件上使用ChatGPT时，必须要提供API Key。
