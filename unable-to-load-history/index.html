ChatGPT Unable to load history（无法加载历史记录），不显示账号信息怎么办？

发表于 2023-11-19 更新于 2023-12-01 分类于 tutorial

最近不少朋友在使用ChatGPT时，登录完账号后，左侧提示Unable to load history（无法加载历史记录），不显示账号信息，也无法问ChatGPT问题。这个问题的原因是什么呢？解决方法又是什么呢？今天ChatGPT中文网就分享下遇到这个问题应该怎么办。

Unable to load history的原因

ChatGPT Unable to load history（无法加载历史记录）错误的原因是没有成功加载ChatGPT网站，历史记录没法显示，账号信息没法显示，也没法正常提问ChatGPT，如下图：

一般来说，Unable to load history（无法加载历史记录）无法成功加载ChatGPT网站可能是以下两个原因：

  • ChatGPT服务器资源过载（一般在ChatGPT推出新功能时可能会出现）
  • 你的网络不稳定（梯子不行）

最近不少人在用ChatGPT时都遇到了ChatGPT Unable to load history（无法加载历史记录），可能是ChatGPT对网络的要求变高了。

Unable to load history的解决方法

如果你也遇到了这个无法加载历史的问题，首先去确认下是否是ChatGPT服务器资源过载导致的，可以在这个网站看下OpenAI服务状态：https://status.openai.com

不过一般来说ChatGPT服务器资源过载的情况还是比较少发生的，这个错误的原因大概率还是使用的网络不稳定，所以可以尝试换个节点，重新刷新，看能否成功加载资源。

（你也可以加入ChatGPT交流群，看看到底是不是因为ChatGPT官方的问题：ChatGPT交流群）
