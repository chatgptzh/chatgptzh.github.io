不能用VPN访问ChatGPT了？访问ChatGPT提示"如果您正在使用VPN，请尝试将其关闭"怎么办

发表于 2023-06-28 分类于 tutorial

ChatGPT交流群中最近不少朋友在问：ChatGPT是不是不能用VPN访问了？打开VPN代理访问ChatGPT时报错Sorry, you have been blocked. If you are using a VPN, try turning it off.（抱歉，您已被屏蔽。如果您正在使用VPN，请尝试将其关闭。）。今天ChatGPT中文网就分享下这个问题的原因和解决方法。

ChatGPT被检测出在使用VPN

不少朋友在开了VPN代理后，再访问ChatGPT（https://chat.openai.com）会报错Sorry, you have been blocked. If you are using a VPN, try turning it off.（抱歉，您已被屏蔽。如果您正在使用VPN，请尝试将其关闭。）：

其实这个问题跟之前分享的Sorry, you have been blocked（抱歉，您已被屏蔽）是同一个问题，只不过ChatGPT在上面加了一句：If you are using a VPN, try turning it off.，因为本来ChatGPT封禁的就都是代理IP或者IDC的IP，所以遇到这个错误默认就是你正在使用VPN访问ChatGPT，加这句描述也没毛病。

如何用VPN使用ChatGPT

那么是不是所有VPN都不能使用ChatGPT了？在国内就没法用ChatGPT了？

当然不是！我们知道，ChatGPT支持的国家只有163个，我们如果想在不被支持的国家或者地区使用ChatGPT，只能使用VPN改变我们的IP地址。

所以，遇到这个问题的解决方法就是找到一个支持ChatGPT的VPN。例如搬瓦工的Just My Socks，据ChatGPT中文网所知，它是确保能访问ChatGPT的：《在Just My Socks网络上使用ChatGPT》。
