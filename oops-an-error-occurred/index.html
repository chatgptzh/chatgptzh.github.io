发表于 2024-08-08 分类于 tutorial

可以正常访问ChatGPT，也可以登录ChatGPT账号，但是一发送消息给ChatGPT，就报错Oops, an error occurred（出现错误），并且重试了很多次也解决不了。今天ChatGPT中文网就分享下遇到这个问题应该怎么办。

ChatGPT an error occurred问题原因

能正常访问ChatGPT官网，也能成功登录ChatGPT账号，但是给ChatGPT发消息后，就报错Oops, an error occurred（出现错误），如下图：

虽然图上有【Try again】的按钮，但是试了很多次还是同样的错误，无法发消息。这次这个问题的原因是网络环境问题。

ChatGPT an error occurred解决方法

根据ChatGPT交流群中用户的解决经验，可以1）先用无痕模式重新登录账号再提问试试是不是cookies或者缓存的问题；2）再换一个浏览器试试是不是浏览器的问题，有朋友就是从Chrome换成Firefox后就解决这个问题了。

当然，节点也很重要，不干净的节点是用不了ChatGPT的，换了浏览器后，也要试试还用不同的节点看看能不能正常使用ChatGPT。可以问问自己购买的工具的客服看看他们有没有经验。
