发表于 2023-04-12 更新于 2023-05-11 分类于 news

人工智能大火，我们在哪里都会看到ChatGPT相关的新闻或者视频，这些报道中总会有一些专业名词，听的人雨里雾里的，今天ChatGPT中文网就先给大家科普下这些专业名词的解释。

OpenAI

OpenAI是美国的一个人工智能研究实验室，由非营利组织OpenAI Inc和其营利组织子公司OpenAI LP所组成。OpenAI进行AI研究的目的是促进和发展友好的人工智能，使人类整体受益。

GPT

Generative Pre-trained Transformer（GPT）是由OpenAI推出的一个预训练语言模型，能出色的处理非常复杂的NLP任务，如文章生成，代码生成，机器翻译，Q&A等。

ChatGPT

ChatGPT是一个人工智能聊天机器人程序，于2022年11月推出，目前是一个网站，用户在电脑或者手机上打开网页就可以直接对话，完全免费，注册即可使用。

ChatGPT可以做的事很多，包括但不限于生成文章、协助写代码、翻译、问答等。

GPT-3.5

GPT-3.5是GPT的一个架构版本，目前ChatGPT默认就是使用GPT-3.5架构。

GPT-4

GPT-4 是2023年3月最新发布的全新版本的GPT架构，并且是一个多模态大模型，支持接受图像和文本输入，输出文本。

ChatGPT Plus

ChatGPT Plus是ChatGPT的会员版本，在注册ChatGPT账号后，用户可以花费20美元/月的价格升级为ChatGPT Plus。ChatGPT Plus可以使用GPT-4。

API

这里的API主要是指ChatGPT API，也就是ChatGPT人工智能模型的接口。我们在使用一些工具时都会看到“ChatGPT API secret key”的填入要求，这里说的就是ChatGPT API。

ChatGPT API是收费的， 目前的定价为是$0.002/1K tokens，大约750个词。
